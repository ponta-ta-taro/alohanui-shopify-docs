# Genspark 改善マニュアル

## このドキュメントについて

このファイルは、Genspark AIによる「アロハ縫茅ヶ崎」サイトの改善提案を記録するためのものです。

## 📌 使い方

1. Gensparkから受け取った改善提案をこのファイルに転記してください
2. 各改善項目は見出しで分類してください
3. 実装時は該当部分を参照し、`docs/prompts/` のプロンプトに詳細を記載してください

## 改善提案の構造（推奨）

各改善項目について以下の形式で記録することを推奨します：

```markdown
### [改善項目名]

**現状の課題**:
- 具体的な問題点を箇条書き

**改善提案**:
- Gensparkが提案した内容

**期待される効果**:
- コンバージョン率向上
- ユーザー体験改善
- など

**実装の優先度**: 高 / 中 / 低

**関連プロンプト**: `docs/prompts/XX-xxx.md`
```
